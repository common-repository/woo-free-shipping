== Description ==
This woocommerce based plugin helps you make flat shipping free for users having specific quantity of orders in their cart. 
Let say, you want to offer your clients free shipping on orders having 3 or more items in their cart, than using this plugin you will be able to set value 3 in Woocommerce Shipping settings page and you are all set to go.

== Installation ==
1. Upload `irb-free-shipping.php` to the `/wp-content/plugins/` directory 
2. Activate this plugin through the \'Plugins\' menu in Wordpress
3. Go to Woocommerce > Settings > Shipping > Shipping Options and adjust the value against \"Offer Free Shipping\" field.
4. You are all set and good to go.
